# tiktok-total
Script desarrollado en python para la gestión de cuentas en TikTok mediante scrapping: descargas de colecciones, desmarcado de favoritos, comprobaciones de followers y followings, realizar unfollowings masivos, etc.

Iré gestionándolo por módulos que se invocarán desde menú principal.
